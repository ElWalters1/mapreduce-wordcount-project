# Hadoop MapReduce Word Frequency Analysis

## Overview
This project uses Hadoop MapReduce to analyze word frequency in Shakespeare’s works.

## Technologies
- Hadoop
- Python (2.7)
- Cloudera VM

## Files
- mapper.py
- reducer.py

## How to Run
1. Upload data to HDFS
2. Run Hadoop streaming job
3. View results

## Features
- Stopword removal
- Word frequency counting
